# servlet-jsp-
适合新手的练手项目
主要实现了登录注册忘记密码找回的功能,忘记密码找回是通过注册时留下的邮箱给你的邮箱发送账号密码的形式实现。
用的技术有： jsp servlet 实现过程中二选一(数据库连接池阿里的DRUID，简单版本的连接数据库) 前端基本知识
